# Requirements for Multi-Agent Newsroom

This project requires the following dependencies:

1. **flask**: For creating the web application framework.
2. **requests**: For making HTTP requests to external services.
3. **numpy**: For numerical computations, especially during data processing.
4. **pandas**: For data manipulation and analysis.
5. **beautifulsoup4**: For parsing HTML and XML documents.
6. **scikit-learn**: For machine learning functionalities.
7. **matplotlib**: For plotting graphs and visualizing data.
8. **pytest**: For running unit tests.

These packages can be installed using pip:
```bash
pip install -r requirements.txt
```
